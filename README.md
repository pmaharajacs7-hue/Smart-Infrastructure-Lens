# 🛡️ Smart Infrastructure Lens

Real-time monitoring dashboard for electrical grid and water pipeline infrastructure, built with Streamlit and machine learning.

## Features

- **Electrical Grid** — Transformer, Substation, and Power Line monitoring
  - Isolation Forest anomaly detection
  - IEEE/ANSI/NESC rule engine (C57.91, C84.1, Rule 232, IEEE 738)
  - Diagnostic layer: identifies primary fault driver with sigma deviation
- **Water Pipeline** — Station → J1 → J2 → J3 → Home network
  - Random Forest classifier (Normal / Moderate / Critical)
  - Pressure and flow sensor tracking across junctions
- **Live map** — Plotly Scattermapbox with real-time color updates (no page reload)
- **Sensor charts** — History plots per asset, status distribution donuts
- **3-page navigation** — Home → Map → Asset Detail with back buttons

## Project Structure

```
app.py                  # Main Streamlit app (all 3 pages)
engine.py               # GridHealthEngine: Isolation Forest + IEEE rules
train.py                # Pipeline RF training script
simulate.py             # EPANET/WNTR water network simulation
database.py             # Supabase connector (grid_analytics table)
model/
  pipeline_model.pkl    # Trained pipeline RF model
  scaler.pkl
  features.pkl
data/
  pipeline_dataset.csv
  transformers_v5.csv
  substations_v5.csv
  powerlines_v5.csv
network/
  water_network.inp     # EPANET network definition
```

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Model Training

```bash
# Electrical — trains Isolation Forest, saves pkl files
python engine.py

# Pipeline — trains Random Forest classifier
python train.py
```

## Database (Supabase)

Create a `grid_analytics` table in Supabase:

```sql
CREATE TABLE grid_analytics (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    timestamp TIMESTAMPTZ NOT NULL,
    asset_id TEXT NOT NULL,
    asset_type TEXT NOT NULL,
    overall_status TEXT NOT NULL,
    oil_temp_c FLOAT, load_pct FLOAT, vibration_um FLOAT,
    sf6_pressure_bar FLOAT, busbar_temp_c FLOAT,
    busbar_delta_t FLOAT, voltage_stability_pu FLOAT,
    current_amps FLOAT, current_pct_rated FLOAT,
    ground_clearance_ft FLOAT,
    ai_anomaly BOOLEAN DEFAULT FALSE,
    ai_score FLOAT,
    trigger_source TEXT,
    status_details TEXT
);
```

Add your credentials to `database.py` and re-enable the import in `app.py`.
